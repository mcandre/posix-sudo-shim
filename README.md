# posix-sudo-shim: a workaround for UNIX systems that lack full sudo

# EXAMPLE

```console
$ sudo echo 'all ur base r belong to us' >/flag.txt
$ cat /flag.txt
all ur base r belong to us
```

# INSTALL

```console
$ su root -c 'sh install.sh'
```

# UNINSTALL

```console
$ su root -c 'sh uninstall.sh'
```
